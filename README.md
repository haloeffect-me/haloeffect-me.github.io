# haloeffect.github.io

Static GitHub Pages day planner for `haloeffect.github.io`.

## What it does

- lets you search for places in the browser
- adds stops to a day plan
- reorders stops into an itinerary
- generates Citymapper links between each adjacent pair of stops
- saves a plan locally per browser username
- shares itinerary text to WhatsApp on mobile-friendly browsers

## Notes

- the sign-in flow is local to the browser and device
- there is no backend or secure password handling in this static version

## Publish

1. Create or rename the repository to `haloeffect.github.io`.
2. Commit these files to the default branch.
3. In GitHub, enable Pages from the repository root if it is not already enabled.

## Files

- `index.html`: planner UI and page structure
- `styles.css`: layout, typography, and responsive styling
- `script.js`: place search, stop management, and Citymapper link generation
